Separators: Horizontal & Vertical Rules (stylish)

This demo shows a fancy horizontal rule and a vertical rule used to separate content areas. Resize the window to see the vertical rule flip to a horizontal bar on small screens.

Horizontal separator (use <hr>)

Section A

Content for the left column — useful for an intro, summary, or navigation. Use a vertical rule to give a clear visual separation when laying out side-by-side content.

Section B

Content for the right column — detail, form, or additional information. On mobile the vertical rule becomes a horizontal divider so the layout remains readable and accessible.

Tip: use an actual <hr> for semantic horizontal rules. For vertical separators, a styled <div> or CSS pseudo-element is common.
